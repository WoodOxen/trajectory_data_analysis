# Trajectory Data Analysis and Preprocessing

## About

This directory gives a detailed data analysis of several large and sound vehicle trajectory dataset.

## Intro. to the Datasets

### HighD

#### Availability

According to the license, this dataset cannot be distributed on public. Anybody that feels like to use this dataset needs to apply for a link on [highD](https://www.highd-dataset.com/#download).

It is expected to take about one week to get the approval and link. The link will be invalid in one week.

#### Citation

